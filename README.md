# Kanban-board

## Libraries used in this project

- [Vue]
- [Vuex]
- [Vue-router]
- [Vee-validate]
- [Tailwind]
- [Font Awesome]
- [Momentjs]
- [Firebase]

Took the theme switch here:
   https://github.com/huphtur/tailwind-theme-switcher

npm i
npm run dev
